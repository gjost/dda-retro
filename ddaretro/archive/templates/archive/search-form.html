Help With Search

Search Tips

  • To do a simple search, you can just type a few keywords in the box; but you will only get back items that match all of the words you have entered. If you would like to do a more complicated search, see the "Advanced Searching" section below.
  • Searching is not case-sensitive so you do not need to capitalize terms. The order of the keywords doesn't matter either.
  • The current search system only looks for matches with individual video segments, photos, documents and newspapers. For now, it does not search collection descriptions, narrator biographies or the full text of interview transcripts.
  • The list of items in the search results is not ranked. However, you can sort or use the filters to narrow and refine your results.
  • Enter the Densho ID (e.g., denshopd-i151-00046) of a primary source item from the public website to find the original archive object.

Advanced Searching

You can perform more sophisticated searches by using boolean operators (AND, OR, NOT, NEAR), grouping words/phrases using parentheses or quotes, and using the wild card character "*". Here are some example searches:

farm bellevue
finds items containing both 'farm' and 'bellevue'
farm and bellevue
same as above
farm or bellevue
finds items containing 'farm' or 'bellevue'
"tom matsuoka"
finds items containing the exact phrase "tom matsuoka" -- this would not match a document with the phrase 'tom j. matsuoka', for example
farm not bellevue
finds items containing 'farm' but not 'bellevue'
(farm not bellevue) and matsuoka
finds items containing 'matsuoka', plus 'farm' but not 'bellevue'
matsu*
finds items containing words starting with 'matsu' -- such as 'matsumoto' and 'matsuoka'. The wildcard only works at the end of a word or phrase, for example, matsu* or matsuo*, but not matsu*ka
